LiquidityPlus v5 — промокоды, подарочный VIP, торговый дневник и отчёты

Команды администратора:
/promocreate CODE DAYS MAX_USES
/giftvip USER_ID DAYS
/report day
/report week
/report month

Команды пользователя:
/promo CODE
/journaladd BTCUSDT LONG 2.5 комментарий
/journal

Автоматические отчёты отправляются владельцу в 20:00 UTC:
- ежедневно;
- по воскресеньям за неделю;
- 1 числа за последние 30 дней.

Новая база v5_features.db создаётся автоматически. Старые базы не удаляются.
