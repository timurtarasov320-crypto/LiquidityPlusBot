Автоматическое сопровождение сигналов

Что работает:
- Проверка цены через публичный OKX API каждые 20 секунд.
- Автоматические уведомления TP1, TP2, TP3 и Stop Loss.
- После TP1 Stop Loss автоматически переносится в безубыток.
- Если цена возвращается к входу, сигнал закрывается в безубыток.
- Поддерживаются цены с пробелами и диапазоны с -, – или —.
- Автоматические события синхронизируются с /tracking.

Для теста создайте новый сигнал через /admin.
Старые сигналы без записанных получателей не смогут отправить уведомления пользователям.
